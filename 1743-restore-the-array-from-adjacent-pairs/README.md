<h2><a href="https://leetcode.com/problems/restore-the-array-from-adjacent-pairs/">1743. Restore the Array From Adjacent Pairs</a></h2><h3>Medium</h3><hr><div style="border-color: rgb(91, 119, 134) !important;"><p style="border-color: rgb(91, 119, 134) !important;">There is an integer array <code style="background-color: rgb(20, 28, 32) !important; color: rgb(183, 198, 205) !important; border-color: rgb(83, 109, 121) !important;">nums</code> that consists of <code style="background-color: rgb(20, 28, 32) !important; color: rgb(183, 198, 205) !important; border-color: rgb(83, 109, 121) !important;">n</code> <strong style="border-color: rgb(91, 119, 134) !important;">unique </strong>elements, but you have forgotten it. However, you do remember every pair of adjacent elements in <code style="background-color: rgb(20, 28, 32) !important; color: rgb(183, 198, 205) !important; border-color: rgb(83, 109, 121) !important;">nums</code>.</p>

<p style="border-color: rgb(91, 119, 134) !important;">You are given a 2D integer array <code style="background-color: rgb(20, 28, 32) !important; color: rgb(183, 198, 205) !important; border-color: rgb(83, 109, 121) !important;">adjacentPairs</code> of size <code style="background-color: rgb(20, 28, 32) !important; color: rgb(183, 198, 205) !important; border-color: rgb(83, 109, 121) !important;">n - 1</code> where each <code style="background-color: rgb(20, 28, 32) !important; color: rgb(183, 198, 205) !important; border-color: rgb(83, 109, 121) !important;">adjacentPairs[i] = [u<sub style="border-color: rgb(83, 109, 121) !important;">i</sub>, v<sub style="border-color: rgb(83, 109, 121) !important;">i</sub>]</code> indicates that the elements <code style="background-color: rgb(20, 28, 32) !important; color: rgb(183, 198, 205) !important; border-color: rgb(83, 109, 121) !important;">u<sub style="border-color: rgb(83, 109, 121) !important;">i</sub></code> and <code style="background-color: rgb(20, 28, 32) !important; color: rgb(183, 198, 205) !important; border-color: rgb(83, 109, 121) !important;">v<sub style="border-color: rgb(83, 109, 121) !important;">i</sub></code> are adjacent in <code style="background-color: rgb(20, 28, 32) !important; color: rgb(183, 198, 205) !important; border-color: rgb(83, 109, 121) !important;">nums</code>.</p>

<p style="border-color: rgb(91, 119, 134) !important;">It is guaranteed that every adjacent pair of elements <code style="background-color: rgb(20, 28, 32) !important; color: rgb(183, 198, 205) !important; border-color: rgb(83, 109, 121) !important;">nums[i]</code> and <code style="background-color: rgb(20, 28, 32) !important; color: rgb(183, 198, 205) !important; border-color: rgb(83, 109, 121) !important;">nums[i+1]</code> will exist in <code style="background-color: rgb(20, 28, 32) !important; color: rgb(183, 198, 205) !important; border-color: rgb(83, 109, 121) !important;">adjacentPairs</code>, either as <code style="background-color: rgb(20, 28, 32) !important; color: rgb(183, 198, 205) !important; border-color: rgb(83, 109, 121) !important;">[nums[i], nums[i+1]]</code> or <code style="background-color: rgb(20, 28, 32) !important; color: rgb(183, 198, 205) !important; border-color: rgb(83, 109, 121) !important;">[nums[i+1], nums[i]]</code>. The pairs can appear <strong style="border-color: rgb(91, 119, 134) !important;">in any order</strong>.</p>

<p style="border-color: rgb(91, 119, 134) !important;">Return <em style="color: rgb(234, 238, 241) !important; border-color: rgb(91, 119, 134) !important;">the original array </em><code style="background-color: rgb(20, 28, 32) !important; color: rgb(183, 198, 205) !important; border-color: rgb(83, 109, 121) !important;">nums</code><em style="color: rgb(234, 238, 241) !important; border-color: rgb(91, 119, 134) !important;">. If there are multiple solutions, return <strong style="border-color: rgb(91, 119, 134) !important;">any of them</strong></em>.</p>

<p style="border-color: rgb(91, 119, 134) !important;">&nbsp;</p>

<strong style="border-color: rgb(83, 109, 122) !important;">Output:</strong> [1,2,3,4]
