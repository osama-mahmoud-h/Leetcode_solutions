<h2><a href="https://leetcode.com/problems/smallest-range-covering-elements-from-k-lists/">632. Smallest Range Covering Elements from K Lists</a></h2><h3>Hard</h3><hr><div style="border-color: rgb(91, 119, 134) !important;"><p style="border-color: rgb(91, 119, 134) !important;">You have <code style="background-color: rgb(20, 28, 32) !important; color: rgb(183, 198, 205) !important; border-color: rgb(83, 109, 121) !important;">k</code> lists of sorted integers in <strong style="border-color: rgb(91, 119, 134) !important;">non-decreasing&nbsp;order</strong>. Find the <b style="border-color: rgb(91, 119, 134) !important;">smallest</b> range that includes at least one number from each of the <code style="background-color: rgb(20, 28, 32) !important; color: rgb(183, 198, 205) !important; border-color: rgb(83, 109, 121) !important;">k</code> lists.</p>

<p style="border-color: rgb(91, 119, 134) !important;">We define the range <code style="background-color: rgb(20, 28, 32) !important; color: rgb(183, 198, 205) !important; border-color: rgb(83, 109, 121) !important;">[a, b]</code> is smaller than range <code style="background-color: rgb(20, 28, 32) !important; color: rgb(183, 198, 205) !important; border-color: rgb(83, 109, 121) !important;">[c, d]</code> if <code style="background-color: rgb(20, 28, 32) !important; color: rgb(183, 198, 205) !important; border-color: rgb(83, 109, 121) !important;">b - a &lt; d - c</code> <strong style="border-color: rgb(91, 119, 134) !important;">or</strong> <code style="background-color: rgb(20, 28, 32) !important; color: rgb(183, 198, 205) !important; border-color: rgb(83, 109, 121) !important;">a &lt; c</code> if <code style="background-color: rgb(20, 28, 32) !important; color: rgb(183, 198, 205) !important; border-color: rgb(83, 109, 121) !important;">b - a == d - c</code>.</p>

<p style="border-color: rgb(91, 119, 134) !important;">&nbsp;</p>

<strong style="border-color: rgb(83, 109, 122) !important;">Output:</strong> [20,24]
<strong style="border-color: rgb(83, 109, 122) !important;">Explanation: </strong>
